О программе

Loginom — платформа для решения большого спектра бизнес-задач, требующих обработки больших объемов данных, реализации сложной логики и применения методов машинного обучения

Возможности Loginom

Порог входа

Платформа нетребовательна в плане "железа", так как существует браузерная версия, которая уступает в скорости обработки информации (по сравнению с дескопной), но есть оптимальным решением для слабых ПК. Так же визуальное проектирование снижает время на обучание пользователя и не требует знания ЯП.

Интеграция

Программа предоставляет пользователю всевозможные инструменты для решения задач, вплоть до уже готовых решений (скоринговые системы, системы принятий решений, очистка данных, нейронные сети). Публикация веб-сервисов позволяет создать компонент API для большого проекта.

Готовые компоненты        Публикация веб-сервисов  

"Deep Learning"

Платформа имеет множество инструментов для построения кластеров, нейронных сетей и регрессий, а так же их комбинаций для решения задач основаных на больших данных, при этом скорость выполнения существенно выше чем у аналогов

Представления результатов

На любом участке выполнения программы можно создать визульный слепок данных выбраного формата, при этом данные можно "подать" в блок отвечающий за создание отчетов, что позволяет документировать наблюдения без вмешательства в процесс
